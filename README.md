# Cloud Notes API

A simple REST API to create, fetch, and delete notes. Notes are persisted in **DynamoDB** and the API is deployed on **AWS EC2**.

## Tech Stack
- Python 3.9+ (Flask)
- Boto3 (AWS SDK for Python)
- DynamoDB (AWS)
- EC2 (AWS)
